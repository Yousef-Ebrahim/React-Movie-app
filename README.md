# 🚀 React Simple Starter

A clean and simple React starter template built with:

- ⚡ Vite
- ⚛️ React
- 🎨 Tailwind CSS
- 🌼 DaisyUI
- 🌐 React Router (optional)
- 📡 Axios (ready to use)

---

## 📦 Installation

```bash
git clone https://github.com/your-username/react-simple-starter.git
cd react-simple-starter
npm install
npm run dev
